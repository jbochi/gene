# gene

Parallel Genetic Algorithm in Clojure

Work in Progress

## Usage

`src/uncap/core.clj` has a solver for the uncapacitated facility location problem.

`src/weasel/core.clj` has a solver for the weasel program.

## Running tests

    $ lein test
